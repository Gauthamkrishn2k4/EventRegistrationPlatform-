# Event Registration Platform

This is a functional implementation of an Event Registration Platform built using React.js, Django REST Framework, and SQLite.

## Features

- User Registration
- User Login
- Event Listing
- Event Registration
- View Registered Events
- Django Admin Panel
- React Frontend integrated with Django REST APIs

## Tech Stack

### Frontend
- React.js
- Vite

### Backend
- Django
- Django REST Framework

### Database
- SQLite

## Notes

This is a rough implementation focused on completing the core functionality of the assigned project. The frontend and backend are integrated and the primary features are working successfully.
